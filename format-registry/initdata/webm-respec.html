This specification defines the "webm" [=initialization data=] format for use with the [[[ENCRYPTED-MEDIA]]] [[ENCRYPTED-MEDIA]]. This format is commonly used with the [[[EME-STREAM-WEBM]]] [[EME-STREAM-REGISTRY]].

Format

The format is a single [=Decryption key ID|key ID=] of one or more bytes, as defined by the ContentEncKeyID [[Matroska]] element and [[WebM-Encryption]].
